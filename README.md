## About Me

I'm a Systems Analyst currently pursuing a degree in Systems Engineering at UTN-FRCU. My development focus is on automation, AI integration, and building intelligent agents (both reactive and evolutionary).

I specialize in orchestrating workflows and multi-agent systems by combining local LLMs (Ollama), Python GUIs (PyQt6), and workflow engines like N8N to solve practical problems. My daily driver and preferred environment is Linux.

## Project Portfolio

### Artificial Intelligence & Local LLMs

#### [FreeVi](https://github.com/maximofraisinet/FreeVi)
Generates engaging videos from PDF documents using local AI, Text-to-Speech (Kokoro & VibeVoice), and customizable visual sources.

#### [ChatBotAI-Free](https://github.com/maximofraisinet/ChatbotAI-Free)
A fully offline voice chatbot powered by local LLMs (Ollama), real-time speech recognition (faster-whisper), and high-quality neural TTS. Includes classic chat and hands-free live mode.

#### [ChatAI-Free-Multimodel (CAFM)](https://github.com/maximofraisinet/ChatAI-Free-Multimodel)
A CLI tool that orchestrates a multi-agent debate using local LLMs via Ollama. Multiple AI instances discuss a topic across several rounds to converge on a synthesized consensus answer.

#### [Transcribineitor](https://github.com/maximofraisinet/Transcribineitor)
A tool for transcribing audio and video files using local AI, making content more accessible and searchable.

#### [Test Turing Bot (LangChain)](https://github.com/maximofraisinet/Test-Turing-LangChain)
An experimental chatbot prototype built for university coursework using LangChain to explore prompt engineering for Turing-style tests.

### Intelligent Agents & Algorithms

#### [Evolutive Virtual Pet](https://github.com/maximofraisinet/Evlutive-pet)
A model-based reflex agent developed for an AI university course. Simulates an agent that maintains an internal representation of its environment to survive.

#### [Reactive Vacuuming Agent](https://github.com/maximofraisinet/Reactive-vacuuming-agent)
A PyQt6 desktop application simulating a simple reflex agent cleaning a procedurally generated room with obstacles. Visualizes agent reaction capabilities in real-time.

#### [Tree Solver](https://github.com/maximofraisinet/tree-solver)
An interactive AI desktop application (PyQt6) for visualizing and solving tree search algorithms.

### Automation, Web & Utilities

#### [Paper News](https://github.com/maximofraisinet/Papper-News)
An automated system that extracts, processes, and distributes scientific papers from arXiv. Combines TagUI for web automation and DeepSeek AI for intelligent content processing.

#### [N8N Projects](https://github.com/maximofraisinet/N8N-Projects)
Collection of workflow automation projects showcasing API integration capabilities, scheduling, and process optimization.

#### [Storage Inform](https://github.com/maximofraisinet/Storage-Inform)
A command-line tool that scans the entire Linux filesystem and generates a detailed, readable Markdown report about storage usage (containers, caches, packages, etc.) to help with cleanups.

#### [App-BE](https://github.com/Desarrollo-de-Software-2024-Gadea/App-BE)
Backend application developed with C#, Angular, and ABP Framework, demonstrating enterprise-level application development.
